What We Collect

BeReel processes only the public Instagram or TikTok URL you submit. We request the oEmbed HTML and thumbnail; the thumbnail is stored for ≤ 24 hours, then both the file and its database record are deleted automatically. Server logs (IP + timestamp) are kept seven days for abuse prevention and then purged.

User‑Data Deletion

Because media is auto‑deleted within 24 hours, no further action is normally required. If you need earlier erasure, email us the reel URL and we will remove it within 72 hours.
